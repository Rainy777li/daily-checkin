### admin.html + admin.js
管理员登录后进入的管理后台，包含 5 个模块：

| 模块 | 功能 |
|------|------|
| 📊 仪表盘 | 用户总数 / 今日打卡人数 / 打卡率 / 总打卡记录 / TOP5 排行 |
| 📋 任务管理 | 新建/编辑/启用/禁用/删除 打卡任务（图标/名称/积分/排序） |
| 👥 用户管理 | 用户列表、禁用/解禁（不可禁用管理员） |
| 📈 榜单管理 | 按分类录入/编辑/删除榜单条目，自动计算涨幅 |
| 📸 截图审核 | 查看用户提交截图 → 点击预览 → ✅通过/+5分 或 ❌拒绝 |

### 数据互通
- 管理后台录入的任务 → 用户打卡页自动读取
- 管理后台录入的榜单数据 → 用户榜单页直接显示（替换 mock 数据）
- 截图审核通过 → 用户积分自动增加

### 管理员入口
- app.html 顶部栏 ⚙️ 按钮 → 自动跳转 admin.html
- demo 模式：第一个注册的用户自动为管理员
- 非管理员进入 admin.html 会被弹回
